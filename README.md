# Low-Code Development Platform

Rule Designer is the Eclipse-based development environment for ODM developers. Intelligent assistant for ERPNext that uses semantic search, knowledge graphs, and multi-agent workflows to speed up ERP development. ERPNext Copilot is an AI-powered assistant. XcelForm is a Frappe app that allows you to export data to Excel. Sheets is a Google Sheets connector for Frappe. FastAPI for Frappe. Otto is a Frappe Framework app which will be used for adding intelligent automation capabilities. FlexiRule is a visual rule engineer & orchestration engine. Frappe UI rapidly build modern frontends for Frappe apps. Vue-based AdminLTE dashboard for Frappe and ERPNext. Barista is a Frappe app to automate functional testing. Model Generator is a frappé application to generate models based on Doctypes. Factory de Odoo turns a Product Requirements Document into a complete, multi-module Odoo ERP. DreamFactory is a self-hosted platform providing governed API access to any data source. Titanoboa makes complex workflows easy. Supabase is the Postgres development platform. OpenXava generates a fully functional web application from JPA entities. GeneXus Standard Classes for Java and .NET generators. GeneXus standard functions written in TypeScript. Transpiler & runtime build UI5 apps purely in ABAP. Shesha is an open-source Low-Code development framework specifically for .NET developers. Vesture is a flexible and extensible C# framework designed to accelerate enterprise application development. Lowdefy is an open-source (Apache-2.0) low-code framework that lets you build web apps with YAML or JSON configuration files. ToolJet is an AI-native platform for building and deploying internal tools, workflows and AI agents. Appsmith is an open-source low-code platform. ILLA is a robust open source low-code platform. Budibase is an open-source low-code platform. Erupt is a low-code, full-stack class framework. Obsei is a low code AI powered automation tool

The directory structure includes a specific folder for each phase of development and engineering programs:

- __MBSE__: The root folder Arcadia (ARChitecture Analysis and Design Integrated Approach) is a method of tools dedicated to systems engineering and architecture, supported by the Capella modeling tool.    
  - __CAD__: For computer-aided design.
  - __CAM__: Dedicated to the part manufacturing and assembly process.
  - __CAS__: For simulation of end-to-end functionality and performance analysis.
