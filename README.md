#  Artificial Intelligence for Low-Code Development

Ephemeris System Engineering and requirements management application. ReqTool is a model and a requirements editor for the Eclipse Capra framework. Inference code for CodeLlama models. ZEN Engine is a cross-platform, Open-Source Business Rules Engine (BRE). SudoLang is a programming language designed to collaborate with AI language models. Code2Prompt is a powerful context engineering tool designed to ingest codebases and format them for Large Language Models. PseudoMorpher is a free, open-access AI-powered tool that allows users to convert source code across various programming languages and pseudocode formats. Spoon is an open-source library to analyze, rewrite, transform, transpile Java source code. Modules-Based Interface for Computer Programming. Clara is a forward-chaining rules engine. Rule Designer is the Eclipse-based development environment. Intelligent assistant for ERPNext that uses semantic search, knowledge graphs, and multi-agent workflows. Otto is a Frappe Framework app which will be used for adding intelligent automation capabilities. FlexiRule is a visual rule engineer & orchestration engine. Factory de Odoo turns a Product Requirements Document into a complete, multi-module Odoo ERP. DreamFactory is a self-hosted platform providing governed API access to any data source. Supabase is the Postgres development platform. OpenXava generates a fully functional web application from JPA entities. GeneXus Standard Classes for Java and .NET generators. LLVMSharp is a multi-platform .NET Standard library. Shesha is an open-source Low-Code development framework specifically for .NET developers. Lowdefy is an open-source (Apache-2.0) low-code framework. ToolJet is an AI-native platform for building and deploying internal tools, workflows and AI agents. Appsmith is an open-source low-code platform. ILLA is a robust open source low-code platform. Budibase is an open-source low-code platform. Obsei is a low code AI powered automation tool

The directory structure includes a specific folder for each phase of development and engineering programs:

- __MBSE__: The root folder Arcadia (ARChitecture Analysis and Design Integrated Approach) is a method of tools dedicated to systems engineering and architecture, supported by the Capella modeling tool.    
  - __CAD__: For computer-aided design.
  - __CAM__: Dedicated to the part manufacturing and assembly process.
  - __CAS__: For simulation of end-to-end functionality and performance analysis.
