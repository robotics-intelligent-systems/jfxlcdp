# Low-Code Development Platform

Frappe Framework is a low code web framework for real world applications. Frappe MCP allows Frappe apps to function as MCP servers. Otto is a Frappe Framework app which will be used for adding intelligent automation capabilities to Frappe apps. FlexiRule is a visual rule engineer & orchestration engine for Frappe apps and ERPNext. Frappe Docker images and orchestration for Frappe applications. Frappe app for managing bench. A Command Line Interface for quickly building apps with FrappeJS. Basic Unit Test Orchestrator. Integration tests for ERPNext features written in Cypress. DreamFactory is a self-hosted platform providing governed API access to any data source. Titanoboa makes complex workflows easy.  re-frame is a ClojureScript framework for building user interfaces. Supabase is the Postgres development platform. OpenXava generates a fully functional web application from JPA entities. GeneXus Standard Classes for Java and .NET generators. GeneXus standard functions written in TypeScript. OpenUI5 is a JavaScript UI Framework. Shesha is an open-source Low-Code development framework specifically for .NET developers. Vesture is a flexible and extensible C# framework designed to accelerate enterprise application development. Lowdefy is an open-source (Apache-2.0) low-code framework that lets you build web apps with YAML or JSON configuration files. ToolJet is an AI-native platform for building and deploying internal tools, workflows and AI agents. Appsmith is an open-source low-code platform that streamlines custom application development, deployment, and maintenance. ILLA is a robust open source low-code platform for developers to build internal tools. Budibase is an open-source low-code platform designed to help users build internal tools, workflows, and applications quickly and efficiently. Erupt is a low-code, full-stack class framework. Obsei is a low code AI powered automation tool. 

The directory structure includes a specific folder for each phase of development and engineering programs:

- __MBSE__: The root folder Arcadia (ARChitecture Analysis and Design Integrated Approach) is a method of tools dedicated to systems engineering and architecture, supported by the Capella modeling tool.    
  - __CAD__: For computer-aided design.
  - __CAM__: Dedicated to the part manufacturing and assembly process.
  - __CAS__: For simulation of end-to-end functionality and performance analysis.
